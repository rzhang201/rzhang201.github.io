An Evaluation on the Causes and Health Effects of Obesity

Robert Zhang

Introduction: It is well known that obesity has a large number of well known negative health effects such as heart disease, diabetes, high blood pressure, liver disease, sleep apnea, and certain cancers. The causes of obesity have been linked to diet, sedentary lifestyle, sleep, and environment. Therefore, the increased rates of obesity that do do not seem to be stopping any time soon are a cause for concern. Data collection can be used to identify the size of the problem in this case being obesity and can help determine if a certain programme is working.

Purpose: The purpose of this tutorial is to showcase how you can use pythons various features and libraries in order for the reader to have a better understanding on how to obtain data, and use that data to create linear models. It should be noted that this tutorial is meant to just be a simple example of what you can do and there are honestly plenty of flaws in the methodology. For one since we are going to be do doing it based on mostly the means and averages by State then that really means we only have 50 observations which is just too few. Also in some cases, for some of the independent variables that we will be using such as age the age range is just too small from only 31 to 45 simply because there just was not a table for each year age, only large brackets.

def ridge_grad_descent(X, y, T, alpha, penalty):
    m, n = X.shape # m = #examples, n = #features
    theta = np.zeros(n) # initialize parameters
    f = np.zeros(T) # track loss over time
    for i in range(T):
        # loss for current parameter vector theta
        f[i] = 0.5 * np.linalg.norm(X.dot(theta) - y) ** 2 + penalty * (np.linalg.norm(theta)) ** 2
        # compute steepest ascent at f(theta)
        g = np.transpose(X).dot(X.dot(theta) - y) + 2 * penalty * theta
        # step down the gradient
        theta = theta - alpha * g
    return theta, f
